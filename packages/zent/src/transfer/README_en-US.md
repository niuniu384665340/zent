## Transfer

Double column transfer choice box.

### Guides

The shuttle box list is implemented using the `Grid` component and supports all properties of the `Grid` component.

### API

| Property  | Description       | Type   | Required | Default | Alternative |
| --------- | ----------------- | ------ | -------- | ------- | ----------- |
