## Transfer

双栏穿梭选择框。

### 使用指南

穿梭框列表使用`Grid`组件实现，支持`Grid`组件所有属性。

### API

| 参数      | 说明     | 类型   | 是否必须 | 默认值 | 备选值 |
| --------- | -------- | ------ | -------- | ------ | ------ |
| className | 额外类名 | string | 否       |        |        |
